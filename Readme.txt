If there is a database problem:

Open the DataBase folder
Copy the Data_Base.db file
Paste it into the "\release" directory in the Qt project debugger
The "Application final" folder contains the executable application.

The "QT-Application-final" folder contains the application source code.

To avoid errors, you must choose the "debug_qt_amin" folder as a debugger.


-------


En cas de problème de base de données :

-Ouvrez le dossier DataBase
-Copiez le fichier Data_Base.db
-Collez-le dans le répertoire « \release\ » dans le débogueur de projet Qt

Le dossier « Application final » contient l’application exécutable.

Le dossier « QT-Application-final » contient le code source de l’application.

Pour éviter les erreurs, vous devez choisir le dossier « debug_qt_amin » comme débogueur.
